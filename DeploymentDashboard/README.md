Sales Performance Dashboard adalah aplikasi web berbasis Python yang dikembangkan menggunakan framework **Streamlit**.  
Aplikasi ini bertujuan untuk menampilkan analisis performa penjualan dan segmentasi pelanggan berdasarkan data transaksi penjualan dan analisis RFM (Recency, Frequency, Monetary).

Dashboard ini merupakan hasil *rebuild* dari dashboard awal yang dikembangkan menggunakan **Power BI**, kemudian dikembangkan ulang dan dideploy sebagai aplikasi web untuk memenuhi kebutuhan **Deployment Aplikasi**.

## ğŸ§© Dataset
Aplikasi menggunakan dua dataset utama:

1. **Sales Invoice**
   - Berisi data transaksi penjualan
   - Digunakan untuk analisis pendapatan, produk terlaris, performa cabang, dan tren penjualan

2. **RFM Current**
   - Berisi hasil analisis RFM pelanggan
   - Digunakan untuk segmentasi pelanggan

> Catatan: Dataset yang digunakan pada repository ini merupakan data contoh (sample) untuk keperluan akademik.
